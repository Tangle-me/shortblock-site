Android Setup

Download and install ShortBlock

Works on Samsung, Xiaomi, Huawei, OnePlus, Pixel, Motorola and all other Android phones. The app detects your device and walks you through only the steps you need.

⬇ Download ShortBlock APK

Version 1.0  ·  ~6 MB  ·  Android 8.0+

1

Install the APK file

Tap the downloaded file in your notifications or open it from your Downloads folder. Android will ask permission to install apps from this source — this is a standard Android security prompt for any app not from the Play Store.

What you'll see: "Install blocked" or "For your security, your phone is not allowed to install unknown apps from this source."

Tap Settings in the prompt → toggle Allow from this source → go back → tap Install.

2

Dismiss the Play Protect warning

Google will show a warning saying it doesn't recognise the app. This appears for all apps not distributed through the Play Store — it is not a sign the app is unsafe.

Tap Install anyway (you may need to tap "More details" first to see this option).

3

Unlock app permissions (Android 13 & 14 only)

Android 13 and 14 restrict certain permissions for downloaded apps until you explicitly unlock them. If your phone runs Android 12 or older, skip to Step 4.

To check your version: Settings → About phone → Android version.

If you're on Android 13 or 14:

Settings → Apps → Find "System Block" → tap it → Tap ⋮ (three dots, top right) → Tap "Allow restricted settings" → Confirm
⚠ If you skip this: The accessibility service toggle will appear greyed out in Step 6 and blocking will not work.
4

Open the app and set your PIN

Find System Block in your app drawer and open it.

Enter a PIN of at least 4 digits — this is what you'll use to toggle blocking on and off. Confirm it, then choose a security question and answer in case you forget your PIN.

Tap Activate. The app will now walk you through the remaining steps automatically, detecting your phone model and showing you exactly the right instructions.

5

Grant Device Administrator

A system dialog will appear asking you to activate "Device Administrator". Tap Activate.

This prevents your child from uninstalling the app. Without it, any child who knows how to navigate phone settings could remove it.

6

Grant VPN permission

A system dialog will appear: "System Block wants to set up a VPN connection." Tap OK.

This enables the network filter that blocks TikTok. It does not route your traffic through any external server — the filter runs entirely on the device.

7

Enable the Accessibility Service

This is the main blocking engine. The app will prompt you to open Settings and guide you to the right place for your phone brand.

If you need to find it manually, select your phone brand below:

Settings → Accessibility → Installed services → System Block → toggle ON → Allow
Settings → Accessibility → System Block → Use System Block → Allow
Settings → Accessibility → System Block → ON (scroll past "Downloaded apps" if needed)
Also complete Step 8 (battery) and Step 9 (autostart) — required for Xiaomi devices.
Settings → Accessibility features → Accessibility → System Block → ON
Also complete Step 8 (battery) — required for Huawei devices.
Settings → Accessibility → Accessibility features → Installed services → System Block → ON
Settings → Additional settings → Accessibility → System Block → ON
Settings → Accessibility features → Accessibility → System Block → ON
Settings → Accessibility → Downloaded apps → System Block → ON

Once ON, press back, return to the System Block app, and tap Finish.

8

Disable battery optimisation (Xiaomi, Huawei, OnePlus, OPPO, Vivo only)

Some brands aggressively stop background apps to save battery. This will shut down the blocking service after a few minutes. You need to exempt System Block.

Settings → Apps → Manage apps → System Block → Battery saver → select "No restrictions"
Settings → Battery → App launch → System Block → turn OFF "Manage automatically" → Enable: Auto-launch, Secondary launch, Run in background
Settings → Battery → Battery optimisation → System Block → Don't optimise
Settings → Battery → High background power consumption → add System Block

Samsung, Pixel, Motorola, Sony, Nokia users can skip this step.

9

Enable autostart (Xiaomi / Redmi / POCO only)

MIUI and HyperOS prevent apps from restarting after a reboot unless you allow it explicitly. Without this, blocking stops working every time the phone is restarted.

Settings → Apps → Manage apps → System Block → Autostart → toggle ON

All other brands can skip this step.

iOS Setup

Block TikTok and Shorts on iPhone and iPad

iPhones cannot install APK files. Use one of the two methods below — they work on all iPhones and iPads running iOS 14 or later, with no app installation required.

Recommended

DNS Profile

A configuration file that blocks TikTok and YouTube Shorts at the network level — works in Safari, Chrome, and all other browsers. Installs in under 2 minutes.

⬇ Download Profile
Install steps
  1. Tap "Download Profile" above on the iPhone
  2. A prompt says "Profile Downloaded" — tap Close
  3. Go to Settings → General → VPN & Device Management
  4. Tap ShortBlock under "Downloaded Profile"
  5. Tap Install → enter your iPhone passcode → tap Install again
  6. Tap Done
To remove blocking: Settings → General → VPN & Device Management → ShortBlock → Remove Profile. Protect this with Screen Time passcode (see Option 2).
Built-in · No download

Apple Screen Time

Apple's built-in parental controls. Block TikTok entirely and restrict YouTube Shorts. Works completely offline, PIN-protected, and cannot be bypassed without your Screen Time passcode.

Block TikTok
  1. Settings → Screen Time → turn ON
  2. Set a Screen Time Passcode (different from your phone passcode)
  3. Tap App Limits → Add Limit
  4. Select Social Networking → tap TikTok specifically
  5. Set time to 1 minute per day (effectively blocked)
  6. Toggle Block at End of Limit → ON
Block YouTube Shorts
  1. Screen Time → Content & Privacy Restrictions → ON
  2. Content Restrictions → Web Content → Limit Adult Websites
  3. Under "Never Allow" → tap Add Website
  4. Type: youtube.com/shorts

Troubleshoot

Something not working?

Common issues and fixes.

You need to complete Step 3 — Allow restricted settings. This only applies to Android 13 and 14. Go to Settings → Apps → System Block → ⋮ menu → Allow restricted settings. Then go back to the Accessibility settings and try again.
The Accessibility Service may have been turned off. Go to Settings → Accessibility → System Block and check the toggle is still ON. If your phone is a Xiaomi, Huawei, or OnePlus, also check that battery optimisation is disabled for System Block (Step 8) and autostart is enabled (Step 9 for Xiaomi).
This is a battery optimisation issue, most common on Xiaomi, Huawei, and OnePlus devices. Complete Step 8 and (for Xiaomi) Step 9. If you've already done this and it still happens, go to Settings → Apps → System Block → Battery → select "No restrictions" or "Unrestricted".
Open the System Block app → tap "Forgot PIN" on the login screen → answer your security question → set a new PIN. If you also forgot your security answer, use the emergency reset code. Contact support at admin@tangle-me.com to receive the emergency code after identity verification.

Once System Block's VPN is active, Android prevents other VPN apps from running alongside it. If your child has installed a separate VPN app, they would need to disable System Block's VPN first — which requires uninstalling the app, which requires Device Admin removal, which requires your PIN.
