iOS doesn't allow third-party apps to block other apps. Instead, System Block uses two built-in iOS features together to achieve full protection.

Part 1 of 2

DNS PROFILE

This blocks TikTok and browser-based short-form video at the network level. Install it once and it works silently in the background.

01

Download the profile

Tap the button below on the child's iPhone or iPad. Safari will ask to allow the download — tap Allow.

02

Open Settings

Go to Settings. You will see a new banner at the top: "Profile Downloaded". Tap it.

03

Install the profile

Tap Install in the top right. Enter the device passcode if prompted. Tap Install again to confirm.

04

Done

TikTok and browser-based short-form video is now blocked at network level. The child cannot remove this profile without the device passcode.

Note: This profile blocks TikTok domains entirely and prevents browser access to tiktok.com. It does not block YouTube Shorts inside the YouTube app — that requires Part 2.
Part 2 of 2

APPLE SCREEN TIME

Apple's built-in Screen Time can block YouTube Shorts inside the YouTube app and prevent TikTok from being installed. Set it up on the child's device using your own Apple ID.

01

Open Settings → Screen Time

On the child's device, go to Settings → Screen Time. Tap "Turn On Screen Time" if not already enabled.

02

Set a Screen Time passcode

Tap "Use Screen Time Passcode" and set a PIN different from the device passcode. This prevents the child from changing Screen Time settings.

03

Block TikTok app

Go to App Limits → Add Limit → Social Networking → select TikTok → set limit to 1 minute per day. Or use Always Allowed to whitelist only specific apps.

04

Block YouTube Shorts

Go to Content & Privacy Restrictions → Content Restrictions → Web Content → Limit Adult Websites → add "youtube.com/shorts" to the Never Allow list.

05

Prevent app installs

Go to Content & Privacy Restrictions → iTunes & App Store Purchases → Installing Apps → Don't Allow. Child cannot install TikTok or any other app.

Combined result: DNS profile blocks TikTok at network level + Screen Time blocks the YouTube Shorts feed + App install restriction prevents new social apps. Together this gives near-complete protection on iOS.
